# AlifeGuard: Population Control for STALKER Anomaly

Too many online entities kill performance. AlifeGuard maintains a target count by releasing excess NPCs back to offline simulation through the correct X-Ray release chain. They continue existing in A-Life, just not rendered. Removal starts with the farthest entities. Protected characters are never touched.

[Releases](https://github.com/damiansirbu-stalker/AlifeGuard/releases)

Requires: Anomaly 1.5.3, Modded exes, [xlibs](https://www.moddb.com/mods/stalker-anomaly/addons/xlibs-1001), MCM
